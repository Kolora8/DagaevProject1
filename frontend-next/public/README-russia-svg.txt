Сюда нужно положить файл карты:  russia.svg

Где взять (бесплатно, лицензия позволяет использование):
  1. Откройте  https://www.amcharts.com/svg-maps/?map=russia
  2. Нажмите кнопку загрузки SVG (Download SVG).
  3. Сохраните файл в эту папку под именем  russia.svg
     (полный путь:  site/frontend-next/public/russia.svg)

Важно: у элементов регионов в этом файле id в формате ISO 3166-2,
например  RU-MOW (Москва), RU-SPE (Санкт-Петербург), RU-TA (Татарстан).
Приложение само сопоставит их с данными из базы.

Пока файла нет, приложение показывает упрощённую плиточную карту.
После добавления файла просто обновите страницу.
